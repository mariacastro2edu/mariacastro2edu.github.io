Wise

Designing a new feature

View case study

As part of a D&AD New Blood brief, we developed 'Wise Bags,' an intelligent, travel-based budgeting tool designed to empower digital nomads. Leveraging Wise's existing infrastructure, this solution uses real-time transaction data and AI to help users estimate, plan, and save more effectively for each trip, transforming Wise into a proactive financial companion for managing dynamic lifestyles.
